# 成本表转换工具 - 纯前端版本

这是一个基于 SheetJS + Bootstrap 5 构建的纯前端成本表转换工具，支持完全本地处理。

## 🌐 在线演示

访问地址：https://hlccare.github.io/cost-converter-web/

## 🚀 特性

- ✅ **完全本地处理**：所有转换都在浏览器中完成，无需上传服务器
- ✅ **数据绝对安全**：文件不离开您的电脑，保护隐私
- ✅ **支持常见格式**：支持 .xls 和 .xlsx 格式
- ✅ **智能转换**：自动识别中文序号，智能计算汇总金额
- ✅ **响应式设计**：完美支持手机、平板和电脑
- ✅ **完全免费**：无需注册，无需付费

## 📋 使用要求

1. 现代浏览器（Chrome 60+、Firefox 55+、Safari 11+、Edge 79+）
2. Excel 文件必须包含名为"表 1"的工作表
3. 文件大小不超过 20MB
4. 建议使用 Chrome 或 Edge 浏览器获得最佳体验

## 🎯 使用步骤

1. **上传文件**：点击"选择 Excel 文件"按钮或拖放文件到上传区域
2. **开始转换**：点击"开始转换"按钮
3. **预览结果**：查看转换后的数据预览
4. **下载文件**：点击"下载转换结果"保存 Excel 文件

## 🔧 技术栈

- **前端框架**：Bootstrap 5.3
- **图标库**：Bootstrap Icons
- **Excel 处理**：SheetJS (xlsx)
- **编程语言**：原生 JavaScript (ES6+)
